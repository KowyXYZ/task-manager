# 📝 Task Manager

## Features

- **Multiple Task Management**: Move your tasks from progress to closed.
- **Auto-Save**: Your code is saved in local storage.

## Built With

- `Tailwind`
- `NextJs`
- `Javascript`
- `React-DnD`
